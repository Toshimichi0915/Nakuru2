# Nakuru2
Sends error stacktrace on CraftBukkit/Spigot/Paper server to a Discord channel
config.yml
```
token: "bot access token"
levels:
- WARNING
- SEVERE
channel: "Discord channel ID"
```


Spigotサーバーにおいて出力されるエラーをDiscordに送信します。

config.yml
```
token: "botのアクセストークン"
levels:
- WARNING
- SEVERE
channel: "DiscordのチャンネルID"
```
